Python Dataset Explore

Analisi e visualizzazione di dataset di videogiochi in Python

Cos'è questo progetto?

Python Dataset Explore è uno strumento open source per analizzare e visualizzare dati relativi a videogiochi, scritto in Python. Il progetto permette di:

  • Caricare automaticamente un dataset JSON di videogiochi da GitHub
  • Effettuare analisi statistiche su voti, piattaforme, generi e anni di rilascio
  • Visualizzare i risultati tramite grafici chiari e intuitivi

Il progetto è pensato per studenti, appassionati di dati e chiunque voglia esplorare dataset reali in modo semplice e veloce.

Funzionalità principali

  • Analisi automatica dei dati: calcolo di medie, minimi, massimi, distribuzioni
  • Visualizzazione grafica: grafici a barre e a torta generati con matplotlib
  • Gestione di dataset reali in formato JSON
  • Codice Python ben documentato e facilmente estendibile
  • Documentazione dettagliata e wiki ufficiale

Struttura del progetto

  • data/m.json: Dataset dei videogiochi
  • src/analisi.py: Script per l'analisi statistica
  • src/visualizzazione.py: Script per la creazione di grafici
  • DOCUMENTAZIONE.md: Documentazione tecnica in italiano
  • README.md, LICENSE, SECURITY.md, .gitattributes
  • Wiki ufficiale in inglese (link)

Come si usa?

  1. Scarica o clona il progetto da GitHub
  2. Installa le dipendenze Python:
    • requests (per scaricare il dataset)
    • matplotlib (per le visualizzazioni)
  3. Esegui python src/analisi.py per vedere le statistiche testuali
  4. Esegui python src/visualizzazione.py per generare i grafici

FAQ

Posso usare un mio dataset? Sì, basta sostituire il file data/m.json con il tuo dataset (stesso formato).
Serve una connessione internet? Solo per scaricare il dataset la prima volta. Dopo puoi lavorare offline.
Il codice è open source? Sì, con licenza MIT. Puoi modificarlo e riutilizzarlo liberamente.
Dove trovo la documentazione? Nel file DOCUMENTAZIONE.md e nella wiki ufficiale.
